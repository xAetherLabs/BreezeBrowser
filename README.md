### _how to run_?:
- clone this repo
```bash
git clone https://github.com/xAetherLabs/BreezeBrowser
```
- `cd` into `Frontend`
```bash
cd BreezeBrowser/Frontend
```
- `pnpm` preferred but `npm` if installed also works. install packages
```bash
pnpm install
```
- start the app (development)
```bash
npx expo start
```
